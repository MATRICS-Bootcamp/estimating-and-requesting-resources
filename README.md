# Estimating Resources in HPC

In this tutorial, you'll learn:

* How to plan and estimate resources for your jobs

* Best practices for resource estimation

* What resources might you request on HPC?
* How will you request them?
* Strategies for your first request
* Tools to monitor and fine-tune future requests
